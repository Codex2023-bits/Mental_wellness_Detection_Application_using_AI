## Project Overview

The Mental Wellness Web Application is a secure, full-stack web platform that helps users track their physical and mental wellness journey. Built with Java Servlets and JSP, it provides a private, session-based environment where users can log meditation sessions, track exercise, and receive personalized AI-generated wellness reports.

This project was developed as part of a PBL (Project-Based Learning) assignment at BIT College of Engineering.

## Technology Stack

- **Backend**: Java 17, Jakarta Servlet 6.1, Apache Tomcat 10.1
- **Frontend**: JSP, HTML5, CSS3, Bootstrap 4, Chart.js, Font Awesome
- **Database**: PostgreSQL (Supabase)
- **AI**: Groq API (llama-3.1-8b-instant)
- **Build**: Apache Maven
- **Deployment**: Render (cloud hosting)

## Setup & Running Locally

### Prerequisites
- Java 17+
- Maven (or use the included `mvnw` wrapper)

### 1. Set Environment Variables (PowerShell)

```powershell
$env:DB_URL="jdbc:postgresql://aws-1-ap-south-1.pooler.supabase.com:5432/postgres"
$env:DB_USER="postgres.pbwfzhhuchelilcmtljm"
$env:DB_PASS="your_supabase_password"
$env:GROQ_API_KEY="your_groq_api_key"
```

### 2. Run the Application

```powershell
cd "c:\PBL Projects\TASK3\demo"
.\mvnw.cmd cargo:run
```

App starts at: **http://localhost:8080/demo**

### 3. First-Time Database Setup

Visit once to create all tables and demo users:
```
http://localhost:8080/demo/db-setup
```
